1. DESCRIPTIVE STATISTICS (1-2)
   ├─ Mean (average)
   ├─ Median (middle value)
   ├─ Mode (most frequent)
   ├─ Range (spread)
   ├─ Variance (spread squared)
   ├─ Standard Deviation (typical deviation)
   ├─ IQR (middle 50%)
   └─ Skewness & Kurtosis (shape)

2. PROBABILITY FUNDAMENTALS (3)
   ├─ Basic probability P = Favorable/Total
   ├─ Addition rule (OR)
   ├─ Multiplication rule (AND)
   ├─ Complement rule (NOT)
   ├─ Conditional probability P(A|B)
   ├─ Bayes' theorem
   ├─ Independent vs dependent
   └─ Real-world examples (disease, spam, etc.)

3. PROBABILITY DISTRIBUTIONS (4)
   ├─ Normal (continuous, bell curve)
   ├─ Binomial (discrete, fixed trials)
   ├─ Poisson (discrete, events over time)
   ├─ When to use each
   ├─ 68-95-99.7 rule
   └─ Real-world applications

4. HOW THEY CONNECT
   ├─ Descriptive stats describes existing data
   ├─ Probability tells us likelihood
   ├─ Distributions model expected patterns
